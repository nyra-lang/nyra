Changelog

Spec freeze history and release notes.

v1.40.4 (auto-sync deps — 2026-07-08)

  • Cargo-like auto-sync — nyra run . / nyra build . sync nyra.mod requires before compile: fetch missing packages into .nyra/cache/, and prune packages removed from require
  • nyrapkg sync / bare nyrapkg install — same sync after a manual nyra.mod edit (add or remove)
  • remove_dir_all — recursive delete on Unix now unlinks files correctly (fixes package prune from cache)
  • Docs — project files after init, manual require / remove in packages.html, getting-started / modules / tooling
  • Stable tls native — OS TLS client via libnyra_rt_tls_native.a (macOS Secure Transport · Windows SChannel · Linux OpenSSL). CI conformance fixtures tls_native / tls_openssl

v1.40.3 (TLS backends — 2026-07-08)

  • Bundled rustls HTTPS — default TLS client via libnyra_rt_tls.a; no OpenSSL install for get("https://…")
  • Selectable backends — tls rustls|native|openssl in nyra.mod; pinned as features.tls in nyra.lock (native shipped stable in v1.40.4)
  • Errors — HTTPS failures report real handshake/connect messages via tls_last_error()
  • Docs — net/http, NyraPkg, stdlib TLS sections updated

v1.40.0 (errors & async runtime — 2026-07-03)

  • Official errors — stdlib/error.ny adds Error, context/cause formatting, stack trace hook, and Result<T, Error> helpers.
  • Fallible stdlib wrappers — read_text, write_text, append_text, json_string, json_i32, and json_bool.
  • Official async facade — stdlib/async/mod.ny exposes NyraRuntime_default, NyraRuntime_run_until, sleep_ms_async, and await_i32.
  • Compiler/runtime fixes — ? and LLVM lowering now handle Result<string, Error> / struct error payloads.

v1.39.0 (systems & stdlib — 2026-07-01)

  • random() / random(min, max) — generic integer ChaCha20 CSPRNG builtins (type follows bounds); random_f64(min, max)
  • Metaprogramming docs — comptime + macros + struct JSON synthesis; examples/toolchain/metaprogramming.ny, stdlib/meta/mod.ny
  • Systems features — C union, size_of/align_of, bytes type, SIMD, arena allocator (see CHANGELOG.md)

v1.39.0 (nyrapkg split — 2026-06-30)

  • nyrapkg — standalone package manager in nyra-lang/pkg (written in Nyra): init, add, install, verify, --version, bootstrap, self update, toolchain update
  • Install layout — ~/.nyra/bin/nyrapkg alongside ~/.nyra/bin/nyra
  • nyra pkg trimmed — dependency commands removed from compiler CLI; kept build, prune, c, bind
  • Removed from compiler repo — pkg-registry, stdlib/pkg.ny, in-tree PackageManager app
  • Docs — packages.html, install, getting started, tooling updated
