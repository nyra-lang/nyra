Data Structures

Arrays, vectors, tuples, maps, structs, enums.

Topic guide

Pick a topic:

  • Arrays — fixed size
  • Vectors — growable
  • Tuples — grouped values
  • HashMap — key → value
  • Structs — named fields
  • Enums — fixed variants

Quick tour

Array + tuple + HashMap in one example.

import "stdlib/map"

fn main() {
    let nums = [1, 2, 3]
    print(nums.len())
    let pair = (42, "ok")
    print(pair.0)
    let mut scores = HashMap_str_i32_new()
    scores = scores.insert("ada", 100)
    print(scores.get("ada"))
}
import "stdlib/map"

fn main() -> void {
    let nums: [i32; 3] = [1, 2, 3]
    print(nums.len())
    let pair = (42, "ok")
    print(pair.0)
    let mut scores = HashMap_str_i32_new()
    scores = scores.insert("ada", 100)
    print(scores.get("ada"))
}

Output

3
42
100
