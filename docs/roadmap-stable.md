# Roadmap toward stable releases

Six-phase plan (high level):

1. **Core freeze** — v0.0.1 semver-stable surface ✅
2. **Stable Extended** — `?`, payloads, spawn, Drop ✅ (v0.0.1)
3. **Platform releases** — Linux, macOS, Windows prebuilts ✅
4. **Stdlib hardening** — env, process, DB native drivers, full stdlib in release tarballs ✅ (v0.0.1)
5. **RC stabilization** — conformance 100%, benchmarks, regression (in progress)
6. **Production gates** — async scheduler, traits, LSP maturity (planned)
